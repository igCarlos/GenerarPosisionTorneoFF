# FF Tournament Manager

Web moderna para administrar un torneo de Free Fire por eliminación directa.

## Funciones

- Gestión de hasta 32 participantes.
- Agregar, editar, eliminar, buscar y mezclar jugadores.
- Cuadros automáticos para 2, 4, 8, 16 o 32 posiciones.
- Selección de ganadores haciendo clic en el jugador.
- Avance automático entre rondas.
- Campeón final y estadísticas.
- Guardado automático en `localStorage`.
- Exportación e importación en JSON.
- Diseño responsive para computadoras, tablets y teléfonos.

## Cómo usar

1. Abre `index.html` en el navegador.
2. Agrega los participantes.
3. Presiona **Generar torneo**.
4. Haz clic en el ganador de cada enfrentamiento.

Para publicarlo, puedes subir la carpeta a GitHub Pages, Netlify, Vercel o cualquier hosting estático.
